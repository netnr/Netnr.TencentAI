### 方法命名
- 根据接口/分割，最后是方法名，前面的是类、子类
- 在使用时，看API地址就知道怎么调用方法
- 如：https://api.ai.qq.com/fcgi-bin/ocr/ocr_generalocr
- 调用：`fcgi_bin.Ocr.Ocr_GeneralOcr()`
